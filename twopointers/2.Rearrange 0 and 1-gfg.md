# Intuition
Since the array contains only `0`s and `1`s, we can partition it in-place using the **Two Pointer** technique. One pointer starts from the left and the other from the right. The left pointer skips all correctly placed `0`s, while the right pointer skips all correctly placed `1`s. Whenever the left pointer finds a `1` and the right pointer finds a `0`, we swap them. Repeating this process ensures that all `0`s end up on the left and all `1`s on the right in a single traversal.

# Approach
- Initialize two pointers:
  - `left = 0`
  - `right = len(arr) - 1`
- While `left < right`:
  - Move `left` forward while `arr[left] == 0`.
  - Move `right` backward while `arr[right] == 1`.
  - If `left < right`, swap `arr[left]` and `arr[right]`.
  - Move both pointers inward after the swap.
- Continue until the pointers meet or cross.
- The array is now segregated with all `0`s on the left and all `1`s on the right.

# Complexity
- **Time complexity:**
$$O(n)$$
Each element is visited at most once by either the left or right pointer.

- **Space complexity:**
$$O(1)$$
The array is modified in-place without using any extra data structures.

# Code
```python
class Solution:
    def segregate0and1(self, arr):
        left = 0
        right = len(arr) - 1

        while left < right:
            while left < right and arr[left] == 0:
                left += 1

            while left < right and arr[right] == 1:
                right -= 1

            if left < right:
                arr[left], arr[right] = arr[right], arr[left]
                left += 1
                right -= 1

        return arr
```
