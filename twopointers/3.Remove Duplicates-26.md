# Intuition
Since the array is sorted, all duplicate elements appear next to each other. We can use two pointers:
- A **slow pointer (`i`)** to keep track of the last unique element.
- A **fast pointer (`j`)** to traverse the array.

Whenever the fast pointer finds a new unique element, we move the slow pointer forward and place that unique element at its position.

# Approach
1. If the array is empty, return `0`.
2. Initialize the slow pointer `i = 0`.
3. Traverse the array using the fast pointer `j` from index `1` to `n-1`.
4. If `nums[j]` is different from `nums[i]`:
   - Increment `i`.
   - Copy `nums[j]` to `nums[i]`.
5. After the traversal, the first `i + 1` elements of the array contain all the unique elements.
6. Return `i + 1` as the new length.

# Complexity
- **Time complexity:** `O(n)`
  - We traverse the array only once.

- **Space complexity:** `O(1)`
  - No extra space is used since the array is modified in-place.

# Code
```python
class Solution(object):
    def removeDuplicates(self, nums):
        if not nums:
            return 0

        i = 0

        for j in range(1, len(nums)):
            if nums[j] != nums[i]:
                i += 1
                nums[i] = nums[j]

        return i + 1
```
